# vue-chat

> 本人对vue和原生js理解不是特别透彻，所以该代码可能有写不合理的地方

## 安装所需环境
```
npm install
```

### 启动node建议websocket服务器
```
node server.js
```

### 启动项目
```
npm run serve
```
